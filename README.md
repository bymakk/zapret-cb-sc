Версия zapret с добавленными списками сайтов Chaturbate, Stripchat, Camsoda.

Нажать на зеленую кнопку "Code" и "Download ZIP".
Разархивировать в корень диска C
Запустить одну из стратегий, например general (ALT) от имени администратора.

В браузере выставить Secure DNS - Google DNS т.к. YouTube заблокирован по DNS.

Исходная версия - https://github.com/Flowseal/zapret-discord-youtube
